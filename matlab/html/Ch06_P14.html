Problem 6.14

File: Ch06_P14.m

Consult the documentation for the Matlab function tf to see how to specify a time delay. Then perform a step test simulation for the coupled system to estimate parameters for a first-order plus dead-time model. Compare the new model using another step test simulation.

-------------------------- SOLUTION --------------------------

Step 1. Create a transfer function for the combined system.

Ga = tf([0.8],[0.4 1])*tf([1],[0.4 1]);
Gb = tf([5],[2 3 1],'InputDelay',0.5);

G = Gb*Ga;
step(G);
grid;

Step 2. Create an error function

Gmodel = @(tau,td) tf([4],[tau 1],'InputDelay',td);
t = 0:.1:15;

err = @(tau,td) norm(step(G,t) - step(Gmodel(tau,td),t));

x = fminsearch(@(x) err(x(1),x(2)),[0.4,0]);
tau = x(1);
td = x(2);

hold on
step(Gmodel(tau,td),t);
hold off
legend('Process Model','First Order with Time Delay','Location','SE');

Step 3. Report results.

Gmodel(tau,td)

ans =
 
                      4
  exp(-1.87*s) * -----------
                 2.547 s + 1
 
Continuous-time transfer function.
